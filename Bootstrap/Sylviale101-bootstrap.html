Basic Intro to HTML

Creating Paragraphs

In order to create a body for a paragraph, you should use p and end the sentence with /p. By doing so, this is how you create your first paragraph!

Using Dividers

When using div and ending your code with /div it leaves your sentences and paragraphs with no spaces above or below, making your text squished.
It may seem indifferent to making paragraphs since they are similar to each other. However, if you pay close attention to how the text is cluttered, you'll notice it.

Using the Span Tag

By using the span tag, you can style your words and change the font of them. For example: I like the color plum a lot!

Creating A List

A list in HTML can be created in three different ways: Ordered (ol), Unordered (ul), and descriptive (dl).

Ordered Lists

Here is an example of an ordered list below

  1. Apple Juice
  2. Grape Juice
  3. Pear Juice

Note

Ordered Lists appear numbered

Unordered Lists

Here is an example of a unordered list below

    • Apple Juice
    • Grape Juice
    • Pear Juice

Note

Unordered lists in HTML appear bulleted

Description Lists

Here is an example of a description list below

Apple Juice
Juice Freshly pressed from apples
Grape Juice
Juice obtained from crushed grapes
Pear Juice
Fresh juice created from pears being blended with water

Note

Description lists have three different tags: dl (description list), dt (defined term), and dd (the description of each term).
