Basic Intro to HTML

Creating Paragraphs

In order to create a body for a paragraph, you should use p and end the sentence with /p. By doing so, this is how you create your first paragraph!

Using Dividers

When using div and ending your code with /div it leaves your sentences and paragraphs with no spaces above or below, making your text squished.
It may seem indifferent to making paragraphs since they are similar to each other. However, if you pay close attention to how the text is cluttered, you'll notice it.

Using the Span Tag

By using the span tag, you can style your words and change the font of them. For example: I like the color plum a lot!
